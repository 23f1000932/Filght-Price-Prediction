Forecasting Fare Dynamics

Predicting ticket prices for India's busiest routes

This project investigates the MLP Term 2 (2025) Kaggle Assignment flight price data to model real-world booking behavior. Starting from extensive preprocessing and feature engineering, we built an explainable LightGBM model that captures how travel class, booking window, and airline strategy shape fare fluctuations.

  • Cleaned anomalies and aligned temporal features.
  • Engineered Days Left, class splits, and route embeddings.
  • Benchmarked LightGBM vs. tree ensembles for R^2.
